# c-codebase

Personal library for writing C programs. Used in [kitrofimov/raycaster](https://github.com/kitrofimov/raycaster) and other my projects.

## Features
- Structs:
	- Color
	- Matrix
	- Vector
- Data Structures:
	- DynamicArray
- Utilities:
	- fileUtils
	- memoryUtils
	- stringUtils
	- Logging
